# REVERSE-ENGINEERING - Корпоративный сайт

Современный многостраничный сайт для компании по производству и продаже 3D-печатных изделий и оборудования.

## 🎯 Особенности

- **7 страниц**: Главная, Каталог, Услуги, О компании, Блог, Контакты
- **Светлый современный дизайн**: минималистичный стиль с фирменными цветами (#D32F2F красный, #212121 черный)
- **Полностью адаптивный**: desktop-first подход с мобильным меню
- **Модальная форма заявки**: для всех товаров и услуг
- **Каталог с фильтрами**: по категориям и ценам, переключение вида grid/list
- **Плавная навигация**: smooth scroll и анимации
- **Фиксированный хедер**: удобная навигация на всех страницах

## 🎨 Дизайн-система

### Цвета
- Фон: `#F8F9FA` (светло-серый)
- Карточки: `#FFFFFF` (белый)
- Акцент: `#D32F2F` (красный)
- Текст: `#212121` (черный)

### Типографика
- Заголовки H1: 48-60px, font-weight: medium
- Текст: 16-18px, font-weight: normal
- Sans-serif шрифты (системные)

## 📄 Структура страниц

### Главная страница
- Hero-секция с градиентным фоном и CTA
- Блок преимуществ (3 карточки)
- Популярные товары (3 карточки)
- Отзывы клиентов (3 отзыва)
- CTA-секция с формой заявки

### Каталог товаров
- Боковая панель фильтров (категории, цена)
- Сетка товаров с переключением вида
- Карточки товаров с изображениями, ценами, описаниями
- Кнопка "Заказать" открывает модальное окно

### Услуги
- Описание 3 основных услуг с изображениями
- Блок технологий 3D-печати (FDM, SLA, SLS)
- CTA для консультации

### О компании
- Статистика компании (4 показателя)
- История компании с изображением офиса
- Ценности (4 блока)
- Команда (4 человека)
- Сертификаты и награды
- Портфолио проектов

### Блог
- Фильтры по категориям статей
- Сетка статей (6 статей)
- Карточки с изображениями, датой, автором
- Форма подписки на новости

### Контакты
- Контактная информация (4 блока)
- Форма обратной связи
- Карта офиса (placeholder)
- Список офисов в городах

## 🛠 Технологии

- React 18
- TypeScript
- Tailwind CSS v4
- Lucide React (иконки)
- Unsplash (изображения)

## 🚀 Компоненты

### Переиспользуемые компоненты
- `Header.tsx` - фиксированный хедер с навигацией
- `Footer.tsx` - футер с контактами и навигацией
- `RequestModal.tsx` - модальное окно формы заявки
- `ProductCard.tsx` - карточка товара
- `FilterSidebar.tsx` - боковая панель фильтров

### Страницы
- `Home.tsx` - главная страница
- `Catalog.tsx` - каталог товаров
- `Services.tsx` - услуги
- `About.tsx` - о компании
- `Blog.tsx` - блог
- `Contact.tsx` - контакты

## 📱 Адаптивность

- **Desktop**: полноценное горизонтальное меню
- **Mobile**: hamburger-меню, вертикальная навигация
- **Tablet**: адаптивные сетки (2-3 колонки)

## 🎭 Анимации

- Hover-эффекты на карточках (scale + shadow)
- Плавное появление модального окна (scale-in)
- Smooth scroll при переходах между страницами
- Transition на кнопках и ссылках

## 📝 Использование

Сайт готов к использованию в Figma Make. Все страницы доступны через навигационное меню. Модальная форма заявки открывается при клике на кнопки "Заказать" или "Оставить заявку".

Для интеграции с WordPress/WooCommerce экспортируйте код и подключите соответствующие API для работы с товарами и формами.
